# Daily Summary Skill

24시간 동안 수집된 텔레그램 메시지를 요약하여 사용자 텔레그램으로 전송한다.

## 워크플로우

### 1단계: 메시지 수집

telegram-collector 스킬을 사용하여 최근 24시간 메시지를 수집한다.

```bash
python3 .claude/skills/telegram-collector/scripts/collect_messages.py --hours 24 -o collected_messages/$(date +%Y-%m-%d).md
```

### 2단계: 수집된 메시지 읽기

`collected_messages/YYYY-MM-DD.md` 파일을 읽어 내용을 파악한다.

파일이 너무 크면 주요 키워드로 검색:
- 암호화폐: 비트코인, 이더리움, 코인, 크립토
- 주식: 주가, 실적, 수주, 공시
- 경제: 금리, 환율, 정책

### 3단계: 요약 생성

다음 카테고리별로 핵심 내용을 요약한다:

1. **시장 동향**: 주요 지수, 가격 변동
2. **주요 뉴스**: 중요 이슈, 정책 변화
3. **투자 정보**: 실적, 수주, 공시
4. **액션 아이템**: 주의할 사항, 기회

### 4단계: 텔레그램 전송

```bash
python3 scripts/send_telegram.py "요약 내용"
```

## 요약 형식

```
📊 24시간 시황 요약 (MM/DD HH:MM 기준)

📈 시장 동향
• 핵심 내용 1
• 핵심 내용 2

📰 주요 뉴스
• 뉴스 1
• 뉴스 2

💼 투자 정보
• 정보 1
• 정보 2

⚠️ 주의 사항
• 사항 1

---
총 N개 메시지 중 주요 내용 요약
```

## 가이드라인

- 전체 길이: 2000자 이내 (텔레그램 가독성)
- 중요도 순 정렬
- 출처/시간 포함
- 한국어로 작성
- 수집된 메시지가 없으면 사용자에게 알림
